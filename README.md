# 📝 Blog Website

A full-stack blog website for local development.

## 📁 Project Structure

```
blog-website/
├── frontend/     → HTML, CSS, JS (public-facing blog)
├── backend/      → API server (Node.js / Python)
├── database/     → Schema, migrations, seeds
├── admin/        → Admin dashboard (content management)
└── docs/         → Project documentation
```

## 🚀 Getting Started

1. Clone the repo
2. Copy `.env.example` → `.env` and fill in your values
3. See `docs/setup/` for detailed setup instructions

## 🛠️ Tech Stack

- **Frontend**: HTML, CSS, JavaScript
- **Backend**: Node.js / Express (or Python / Flask)
- **Database**: SQLite (local dev) / PostgreSQL (production)

## 📌 Features

- Blog post listing & detail pages
- About & Contact pages
- Admin panel for creating/editing/deleting posts
- Search functionality
- Responsive design
